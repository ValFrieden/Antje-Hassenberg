🌻 Antje – Van Gogh Galerie 🌻

Diese Website ist fertig für die Veröffentlichung im Internet.

📁 Ordnerstruktur:
├── index.html                  — Hauptseite (Startseite)
├── Bilder/
│   ├── BilderGalerie/          — 21 Fotos (Gemäldegalerie)
│   └── GeschichteEinerFrau/    — 48 Fotos (Geschichte einer Frau)
├── VideosLarge/                — 2 große Videos (~20 MB jedes)
├── VideosSmall/                — 20 kleine Künstler-Videos
└── Musik/                      — 2 MP3-Dateien

🚀 Upload auf einen Webserver:
Einfach den gesamten Inhalt des WebSite-Ordners hochladen (z.B. via FTP).

📝 Hinweise:
- Alle Pfade sind relativ – die Seite funktioniert von überall.
- Für grosse Videos: Statischen Webhosting (z.B. Netlify, Vercel, GitHub Pages) oder eigenen Server empfehlenswert.
- Die Seite ist responsive (Desktop + Mobile).
